Power Apps Formula Floater
==========================

Power Apps Studio の「式入力」エリア（フォーミュラバー）を切り離して、ドラッグ移動・リサイズ可能なフローティングパネルとして表示します。プロパティのタブ表示やピン留め、フィルタ、最小化などで作業性を向上します。

対応ブラウザ: Chrome / Edge (Manifest V3)

インストール（開発者モード）
- Chrome/Edge の拡張機能ページで「デベロッパーモード」を有効化
- 「パッケージ化されていない拡張機能を読み込む」で本フォルダを選択

主な機能
- Alt+Shift+F: フォーミュラバーを自動検出して切り離し
- プロパティのタブ表示: Advancedタブやプロパティのコンボボックスから一覧を抽出してタブ化
- フィルタ入力: タブ横のテキストボックスでプロパティ名を絞り込み
- ピン留めバー: Alt/Ctrl クリックまたはダブルクリックで「コントロール.プロパティ」をピン留め、クリックで即切替
- 最小化/復元: ヘッダーの「Minimize/Expand」で省スペース化（最小化幅は 400px）
- ビジー表示: ピンからの切替中はオーバーレイで進行状況を表示
- ドロップダウン対策: Fluent UI のコールアウトを前面に表示（z-index 調整）

注意
- ページのリロードで切り離し状態やピンはリセット（現状は永続化なし）
- 表示が背面に回る場合は Power Apps 側のスタイル影響の可能性あり

ファイル構成
- `manifest.json` — MV3 マニフェスト
- `service-worker.js` — アクション/ショートカット → コンテンツへ通知
- `content.js` — 切り離し・タブ/フィルタ・ピン留め・高さ調整・切替制御
- `content.css` — フローティング、タブ/フィルタ、ピンバー、オーバーレイのスタイル
